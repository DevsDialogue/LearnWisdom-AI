## Shadcn- dependencies

1. **dropdown-menu**

```bash
npx shadcn@latest add dropdown-menu
npx shadcn@latest add avatar

```

2. **button**

```bash
npm i @radix-ui/react-slot
```

3. **Avatar**

```bash
npm i @radix-ui/react-avatar
```

4. **Dark theme**

```bash
npm i next-themes
